# @holiday-promo/auth

This package handles authentication logic for the Holiday Promo monorepo.

## Features

- Secure authentication flow and helpers
- Integrates with `@holiday-promo/db`
- Uses Zod for schema validation

## Usage

Import the exported helpers to support sign-in/sign-up, user sessions, etc.
